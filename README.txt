This is the data repository for the article entitled:

Visual signaling in Great Tit (Parus major) nestlings under varying daylight intensity

by

Paweł Podkowa1,2,, Szymon M. Drobniak2,3, Adrian Surmacki2
1Institute of Environmental Sciences, Jagiellonian University, Krakow, Poland
2Department of Avian Biology and Ecology, Faculty of Biology, Adam Mickiewicz University, Poznań, Poland.
3Evolution & Ecology Research Centre and School of Biological, Earth and Environmental Sciences, University of New South Wales, Sydney, Australia
 pawel.podkowa@uj.edu.pl


This dataset consists of three Quarto scripts, each providing a step-by-step analysis of the spectral data obtained from nestling beak measurements. The raw spectral data is included in the data.RData file as specs.
The data.RData file contains all the data necessary for processing the analysis.

#### Before you start the analysis make sure to change the working folder path in every load() function to set the proper Working Directory.
